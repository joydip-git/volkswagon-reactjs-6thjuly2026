object in JS
	JS is NOT an OOP language, its an Object-based programming language
	in JS the OOP features are implemented through prototype based programming model

	an object in JS is collection of some keys (properties) and values
you access the members of JS object using dot operator (obj-ref.prop-name) or using indexer (obj-ref['prop-name'])

	JS objects are expandable (dynamic), means you can add additional properties in it later point of time also.

	in every JS object there is a hidden default property known as "proto". It holds a single reference to another object which is treated as its base object.
	That means JS objects supports single and multi-level inheritance BUT NOT MULTIPLE inheritance

	you can copy an object's properties and values to another object by using ... (spread operator). 
	using this you can actually copy properties and values from multiple objects into one. 
	it can override the disadvantage of not having multiple inheritance. 	spread operator does shallow copy of properties and their values.

var targetObj = {...sourceobj1, ...sourceobj2}
