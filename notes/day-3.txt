pass a callback 

function callback(a, b){
	return an integer value
}

1. if the return value is 0 then both a and b are same
2. if the return value is >0 then a is greater than b
3. if the return value is <0 then a is lesser than b

TypeScript: a development mode language
------------------------------------------
1. type inference
2. static type checking
3. strict typification

Note:
strictly typed language: its a strictly typed language: mention the data type. variables are declared always with "var" keyword. if some value is assigned to a variable, then only that type of value can be assigned to the same variable.
