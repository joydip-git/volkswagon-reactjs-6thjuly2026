Developmemt mode web server: (dev server/app server)
---------------------------------------------------------
these web servers hosts your application and serves the static contents (HTML + CSS + JS) on the browser using a certain port number
Requests from dev machine will be processed ONLY

as part of the VS Code you can add "Live Server" extension/plug-in as dev server

it offers you HMR (Hot Module Replacement/Reloading) -> the modules/parts of the app which have been modified/updated, will be reloaded on the browser

JavaScript:
----------------------
its not strictly typed language: no mention of data type. variables are declared always with "var" keyword

loosely-typed language: a variable in JS can store different type of value at different point of time

dynamic language

data type:
-----------------
string, number, Boolean, object

keywords:
------------------
undefined, NaN (Not a number), null, Infinity

operators:
same like any other programming language
arithmetic operator: +, -, *, /, ....
logical operator: and (&&) and or (||)
assignment operator: =, +=, -=,....

control flow statements:
same like any other programming language
if, if...else, while, do...while, switch...case
for, for...of, for...in
continue, break statement
